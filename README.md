# Basic_Calculator

The "Calculator App" is a simple and delightful calculator application developed for Android devices. This app incorporates a visually appealing and heart-themed user interface, with a pink color scheme and a heart-shaped button as the main design element. It allows users to perform basic arithmetic calculations and view the results in a clean and intuitive manner.
Technical Details:

The app is built using Android Studio, utilizing XML for defining the user interface layout (activity_main.xml) and vector drawable (heart_button.xml) for the heart-shaped button. The logic for performing the calculations is implemented in the associated Java code.

skills: android studio, java, xml
